# FICHA BÁSICA — SMART (EUREKA Cluster)

> **Naturaleza:** cluster EUREKA de **fabricación avanzada (advanced manufacturing)**. Financia **proyectos de I+D+i internacionales colaborativos**. El cluster da el **label**; el **dinero lo aporta cada agencia nacional** (en España, **CDTI**). Proceso **PO → FPP**.
> **Sectorial: fabricación avanzada / Industria 4.0.** Año omitido; ciclo típico: PO ~enero, FPP ~abril, label ~junio.

## Identificación
- Cluster EUREKA SMART (Advanced Manufacturing). **1 call al año**, proceso **Project Outline (PO) → Full Project Proposal (FPP)**.
- Instrumento: **label EUREKA + financiación nacional**.
- Comunidad: +140 organizaciones en 21 países EUREKA.

## Elegibilidad (match)
- **Consorcio internacional:** ≥2 partners de **≥2 países EUREKA** (regla general EUREKA).
- Proyecto de **fabricación avanzada** (procesos, tecnologías y sistemas de manufactura) con cooperación y orientación a explotación.
- Cada socio gestiona su **financiación nacional** (CDTI en España).

## Parámetros
| Parámetro | Valor |
|---|---|
| Sector | **Fabricación avanzada / Industria 4.0** |
| Consorcio | Internacional (≥2 países EUREKA) |
| Proceso | **PO → FPP** |
| Financiación | **Nacional** (cada socio); reglas de cada país |
| Ciclo de call | PO ~enero · FPP ~abril · label ~junio (anual; verificar por edición) |

## Evaluación
- Evaluación del cluster (excelencia, impacto, consorcio, ejecución) → **label**; luego financiación nacional.

## Reglas de match / derivaciones
- **Encaja** si: proyecto de **fabricación avanzada / manufactura** con consorcio internacional.
- **No encaja** si: el núcleo no es manufactura (→ software: ITEA; ICT: Celtic; energía: Eurogia), o sin consorcio internacional (→ CDTI nacional).
- **España:** financiación del socio español vía **CDTI** (EUREKA).

## Campos a pedir al cliente
- ¿Proyecto de **fabricación avanzada / Industria 4.0**?
- ¿Consorcio en **≥2 países EUREKA**?
- ¿Orientación a explotación industrial?
- ¿Ha contactado a su agencia nacional (CDTI)?
