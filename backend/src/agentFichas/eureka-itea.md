# FICHA BÁSICA — ITEA (EUREKA Cluster)

> **Naturaleza:** cluster EUREKA de **innovación en software** (software-intensive systems & services), **industry-driven**. Financia **proyectos de I+D+i internacionales colaborativos**. El cluster da el **label**; el **dinero lo aporta cada agencia nacional** (en España, **CDTI**). Proceso **PO → FPP**.
> **Sectorial: software / sistemas y servicios software-intensive** (aplicado a smart mobility, salud, smart cities, energía, industria, ingeniería, seguridad). Año omitido.

## Identificación
- Cluster EUREKA ITEA (software innovation). **1 call al año**, proceso **Project Outline (PO) → Full Project Proposal (FPP)**.
- Instrumento: **label EUREKA + financiación nacional**.
- Comunidad: gran industria, PYMES, startups, academia y organizaciones cliente; 37 países.

## Elegibilidad (match)
- **Consorcio internacional:** ≥2 partners de **≥2 países EUREKA** (regla general EUREKA).
- Proyecto de **software / sistemas software-intensive** con impacto de negocio; orientado a explotación.
- Cada socio gestiona su **financiación nacional** (CDTI en España).

## Parámetros
| Parámetro | Valor |
|---|---|
| Sector | **Innovación en software / software-intensive systems & services** (smart cities, communities, health, mobility, industry, energy, engineering, safety & security) |
| Consorcio | Internacional (≥2 países EUREKA); multi-partner (industria + PYMES + academia) |
| Proceso | **PO → FPP** |
| Financiación | **Nacional** (cada socio); reglas de cada país |
| Calls | **1/año** |

## Evaluación
- Evaluación según criterios de selección del cluster (impacto de negocio, innovación, calidad/consorcio) → **label**; luego financiación nacional.

## Reglas de match / derivaciones
- **Encaja** si: proyecto con **fuerte componente de software** y consorcio internacional, orientado a explotación de negocio.
- **No encaja** si: el núcleo no es software (→ ICT/telecom: Celtic; energía: Eurogia; manufactura: SMART), o sin consorcio internacional (→ CDTI nacional).
- Solape con Celtic-Next: si es **telecomunicaciones/redes**, Celtic; si es **software/servicios software-intensive** transversal, ITEA.
- **España:** financiación del socio español vía **CDTI** (EUREKA).

## Campos a pedir al cliente
- ¿El proyecto tiene **el software como núcleo** de la innovación?
- ¿Consorcio en **≥2 países EUREKA**? ¿industria + PYMES + academia?
- ¿Orientación a explotación/negocio?
- ¿Ha contactado a su agencia nacional (CDTI)?
